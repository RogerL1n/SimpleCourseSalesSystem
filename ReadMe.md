### 模拟售课平台系统


 
系统分成两个部分 : 售课方与顾客方
 
二者在仅在web层上有较大差异,调用接口大同小异
 
故此采用分模组的形式开发,按需引入所需模组
 
实现模组可插拔更替,降低耦合度

前端页面用easyUI和layUi编写
